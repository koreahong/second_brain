# 완벽한 상태가 아닌 상태에서 고객에게 보여준다는 것

## 본 것

내가 트렌드리포트에 기능을 완벽하지 않은 상태에서 ‘이러면 좋지 않을까’ 하는 상태에서 현대쪽에 오픈했다. 그러다보니 오류가 지속적으로 발생하고 계속해서 차질이 생김

## 깨달은 것

완벽하지 않은 상태에서 고객에게 전달하면 절대 안된다. 그렇기 때문에 평소에 준비를 하고 완벽한 상태에서 전달해야 한다. 또한 완벽한 상태에 해야하는 상황이면 기한을 길게 잡거나 예상되는 오류를 알려야 한다.

## 적용할 것

평소에 완벽한 것들을 준비하고 완벽하지 않으면 기한을 길게 세팅하고 예상되는 오류를 알리자
