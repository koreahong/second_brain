# 대시보드 수정할 때 현대 측 분들과 의사소통을 잘 진행하여 해결했다

## 본 것

로그인이 안되는 문제가 발생했는데, 서로 어디서 문제인지 모르는 상황이었고 현대 쪽에 문제인거 같았음

## 깨달은 것

관련된 현대 측 분들에게 구체적인 질문으로 문제해결 실마리를 찾음

## 적용할 것

일단 먼저 들어보고, 문제를 파악하기 위해 한번 더 확인했던 점을 칭찬함. 우리 잘못이 아니라고만 생각하고 보지 않았다면 오류가 발생하는 기간이 길어졌을 것임
