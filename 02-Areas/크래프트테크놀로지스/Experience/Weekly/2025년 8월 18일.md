# 2025년 8월 18일

## 🚩 문제 상황

• 원천 데이터 계약·구매 리스트가 흩어져 관리되고 있어 전체 현황을 파악하기 어려움.
• 각 팀은 자신들이 어떤 데이터를 어떻게 사용하고 있는지조차 명확히 알지 못하는 상황.
• 실제 사용자에게 확인했을 때조차 데이터 사용 내역을 알지 못하는 경우가 많아, 거버넌스 단절이 드러남.

## 🔧 대처 방법

• 팀별 소통을 통해 데이터 계약/구매 관련 자료 수집.
• CFO 미팅을 통해 공식적으로 공유·확인 절차 진행.

## ✨ 성과 / 개선점

• 성과: 원천 데이터의 구매·사용·적재 현황을 정리해 목록화.
• 개선: 팀별 데이터 활용에 대한 인식과 관리 수준이 낮아, 단순 리스트업 이상의 체계화된 관리가 필요.

## 🤝 배운 점

• 실제 데이터를 쓰는 사람들조차 사용 현황을 모르는 경우가 있어,

→ 단순 현황 파악이 아니라 구매리스트를 기반으로 모든 데이터를 일일이 검증해야만 정확한 그림을 그릴 수 있음을 깨달음.

## 💡 한 줄 요약

• 원천 데이터 계약/구매 현황을 정리하며, 데이터 사용 관리가 체계적으로 단절된 문제를 확인했다.

## 역량

#문서화 #의사소통 #문제해결 #운영 체계화

실제로 쓰는 사람들에게 물어봤는데 모르는 상황이라서, 구매리스트를 기반으로 하나씩 전부다 뜯어봐야하는 상황임
