# 2과목

## 📝 내용

3장 투자분석기법

이표채: 매기마다 이자지급. 만기(액면)지급

할인채: 매기마다 이자지급 X

4장

보토주의 가치평가는 가정하에 평가가 들어가는 것

투자수익 = 매매차익 + 배당수익

투자수익률 = 투자수익 / 매수

주식가치 = 배당 / 요구수익률

투자수익 = 투자액 * 요구수익률

5 재무비율

자산 부채 자본- 재무비율

유동자산 1년이내 돈이 들어오는 것

유동자산 = 당좌 + 재고

당좌 = 현금 / 유가증권(시장성 O, 시장성 X) 

현금비율: 당장 현금화할 수 있는 것만 보는 가장 엄격한 기준

유동자산 < 당좌 < 현금 + 시장성 유가증권 순으로 엄격

부채비율 = 부채 / 총자산 (타인자본 + 자기자본) 50% 이하가 바람직함
