# GTM 개념

## 📝 내용

## 개념

- 서비스의 웹 혹은 앱의 태그 관리 시스템으로 서비스에 직접 코드를 심지 않고 사용자가 정의한 태그에 따라 로그를 추출해주는 도구

## 내용

### 구성

- 변수: 로그를 쌓을 때, 로그를 구성한 값들 중에 각각 변수 명으로 매핑해서 전달하는 단위

- 트리거: 로그를 추출하는 유저의 액션 기준

- 태그: 정의한 변수와 트리거를 연동하여 로그를 추출하는 액션을 정의하는 단위

### 변수

- 자바스크립트변수: 웹페이지에서 정의된 전역변수를 그대로 가져올 경우 사용

- 맞춤 자바스크립트: 변수를 그대로 들고 오는 것이 아니라, 사용자가 변수에 대해서 일련의 작업을 진행하고 가져올 경우

- 데이터 영역 변수: datalayer에서 push하는 value명을 정의

### 태그

- 맞춤 HTML: 스크립트를 활용해서 사용자가 정의한 이벤트와 값들을 설정하는 태그

datalayer를 이전에 생성한 변수에 담아서 특정 이벤트를 발생시키는 태그

- GA4 이벤트: GA에 데이터를 전송하는 태그

view_item이라는 이벤트가 발생할 경우 이벤트 매개변수와 매핑하고 GA 데이터를 전송
