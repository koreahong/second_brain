# [기술본부] Data Warehouse Engineer (5년 이상)

## 포지션 상세

DEVSISTERS는 게임을 통해 플레이어들에게 좋은 추억을 선물하고, 더 나아가 세상을 즐겁게 만들 수 있기를 바랍니다.

[팀데브 기술본부는]

데브시스터즈가 게임 산업에서 앞서 나갈 수 있도록 기술적 탁월함을 추구하는 조직입니다.

제품 개발팀이 본질적인 가치에 집중하여 사용자들에게 최고의 즐거움을 전달할 수 있도록 다양한 직군의 전문가들이 모여 필요한 서비스를 개발, 운영하고, 지식과 프로세스를 제공하며, 탄탄한 플랫폼을 구축하여 전사적으로 복잡한 도전 과제를 해결하는 중요한 역할을 담당합니다.

- 서비스 및 데이터플랫폼, 인프라 등을 제공하는 데브시스터즈의 자체 플랫폼(DevPlay)의 안정적인 서비스 운영에 필요한 기능, 서비스, 컨설팅 역량을 보유하고 있습니다.
- 서버, 클라이언트 등 각종 경로로 수집되는 데이터를 데이터 레이크하우스로 가공하여 저장하고, 지표 관리 뿐만 아니라 데이터 분석 컨설팅 및 솔루션을 제공합니다.
- 데이터를 기반으로 한 효과적이고 체계적인 품질 검수를 진행하여 서비스를 지속적으로 개선하고 사용자에게 최고의 사용 경험을 제공합니다.
[데이터인텔리전스 셀은]

데이터인텔리전스 셀은 240여개국, 누적 2억 명 이상의 유저를 보유한 데브시스터즈의 게임 서비스 데이터를 전사 구성원이 효과적으로 활용할 수 있도록 지원하는 역할을 담당합니다.

Data Warehouse Engineer와 Data Analyst로 구성된 데이터인텔리전스 셀은 전사 구성원이 데이터 기반의 인사이트를 통해 고객과 회사에 기여할 수 있도록 데이터를 서비스화하여 제공합니다.

- 데이터의 논리적 흐름과 구조를 관리하여, 데이터의 일관성과 신뢰성을 보장합니다.
- 사업적으로 중요한 데이터를 정의하고 분석하여 비즈니스와 제품 품질 향상에 기여합니다.
- 전사적으로 데이터를 효과적으로 활용할 수 있도록 가이드를 제공합니다.
[데이터인텔리전스 셀이 제공하는 데이터 제품과 서비스는 다음과 같습니다.]

- 전사 통합 데이터 레이크하우스/웨어하우스: 전사 구성원이 신뢰할 수 있는 데이터 소스를 제공하여, 일관성 있는 데이터 기반 의사결정을 지원합니다.
- 데이터 포털 웹사이트: 주요 의사결정권자를 위한 데이터 포털을 운영하여 필요한 데이터를 손쉽게 조회하고 활용할 수 있도록 지원합니다.
- 사업 운영을 위한 데이터 분석: 게임 외 다양한 데이터를 분석하여 효율적인 사업 운영에 필요한 인사이트를 제공합니다.
- 게임별 데이터 웨어하우스: 각 게임의 특성에 맞춰 데이터를 저장하고 최적화하여 성과 분석을 지원합니다.
주요업무Data Warehouse Engineer는• 데브시스터즈 내의 데이터 저장소로서 핵심 역할을 수행하는 데이터 웨어하우스의 구조를 설계하고, 최적화된 운영 방식을 구축합니다.• 매일 10TB+ 단위로 쌓이는 대용량 비정형 데이터를 효율적으로 처리하기 위해 데이터 웨어하우스에 적합한 데이터 모델과 파이프라인을 설계 및 개발합니다.• 변화하는 비즈니스 요구에 맞춰 데이터 모델과 거버넌스를 관리하며, 데이터를 쉽게 활용할 수 있도록 가이드를 제공합니다.[주요 기술/데이터 스택]Cloud Data Warehouse Modeling, Delta Lake, Databricks, Spark SQL, Airflow, AWS[담당업무]• 데이터 웨어하우스의 설계 및 최적화를 통해 성능을 모니터링하고 개선합니다.• 데이터 모델링 및 스키마 설계를 통해 데이터의 정합성과 일관성을 보장하며, 전사 구성원의 데이터 리터러시 향상에 기여합니다.• 데이터 마트 및 대시보드 개발을 통해 필요한 지표와 데이터를 사용자에게 제공합니다.• 사내 데이터 포털 운영 및 개선을 통해 전사 구성원이 데이터를 쉽게 접근하고 활용할 수 있도록 지원합니다.• 신규 제품 런칭 과정에 참여하여, 제품 성과 측정을 위한 데이터 모델을 설계합니다.

자격요건• 관련 경력 5년 이상 또는 이에 준하는 깊이 있는 실무 경험• 복잡하고 최적화된 SQL 쿼리를 이해하고 작성할 수 있는 능력• 데이터 웨어하우스 모델링에 대한 깊이 있는 경험• 데이터 웨어하우스 시스템의 설계, 구축 및 운영을 주도적으로 수행한 경험• 데이터 웨어하우스 구조 설계 시 운영 편의성과 사용 편의성 사이에서 명확한 기준으로 타협점을 찾고 문제 해결에 집중할 수 있는 능력• 데이터 거버넌스 구축 경험 및 데이터를 통한 조직 내 의사결정 지원 경험• BI 툴 또는 OLAP 솔루션 사용 경험 (QuickSight, Tableau 등)과 이를 통한 대시보드 설계/운영 경험• 의사결정자 및 유관 부서와 원활하게 소통하고 복잡한 요구를 조율할 수 있는 능력

우대사항• 클라우드 기반(Databricks, Snowflake 등) 데이터 웨어하우스 개발 및 운영 경험• Spark 및 Airflow 기반의 ETL 설계 및 성능 최적화 경험• Query Plan 해석 및 튜닝 계획 수립을 통한 데이터 처리 최적화 경험• 데이터 표준화, 정합성 및 품질 관리 경험• Python 등 프로그래밍 언어에 대한 이해• 대용량 데이터를 다뤄본 경험• 비정형 데이터를 다뤄본 경험• 게임 데이터를 다뤄본 경험[공통 지원자격]• 한국어 비즈니스 의사 소통 능력• 해외 여행에 결격 사유가 없으신 분[제출서류] *PDF 제출• (필수) 데브시스터즈 및 해당 직무에 대한 지원동기가 포함된 이력서[고용 형태]정규직3개월 수습기간 부여 (수습기간 동안 급여 100%지급)[소속회사]데브시스터즈(주)

혜택 및 복지[데브시스터즈 복지와 혜택]더 나은 생활을 지원합니다.• 유연근로제• 리프레시 유급 휴가 및 휴가비 (3, 5, 7, 9년... 근속 시)더 건강한 삶을 생각합니다.• 건강검진 (본인, 양가 부모, 배우자 전액 지원)• 상해보험 (본인, 양가 부모, 배우자, 자녀 전액 지원)• 사내 피트니스 센터• 사내 레스토랑 및 카페• 스낵바신나고 즐거운 회사를 만들어 갑니다.• 여가 지원 포인트• 생일 및 명절 기프트• 근속 기념 기프트 및 세레머니성장과 몰입을 응원합니다.• 최신 IT 장비 및 업무용 소프트웨어• 업무 능력 향상과 자기계발을 위한 교육 및 세미나• 1:1 화상 어학 교육• 업무용 도서• 월세 지원• 배네핏 카드가족과 함께합니다.• 경조 휴가 및 경조금• 상조 프로그램• 자녀 입학 기프트• 출산 축하 기프트• 태아검진 유급 반차제도• 난임 유급 휴가
