직무정보채용공고 상세 목록회사명직무명데이터 엔지니어근무형태정규직채용구분경력근무지서울직무소개• 산재되어 있는 정형/비정형 데이터를 목적에 맞게 수집/제공/활용하는 데이터플랫폼 구축/운영• 실시간 데이터를 수집하고, 외부통신을 통한 실시간 데이터 전송 파이프라인 개발/운영• 협력광고, 리테일미디어 등 광고플랫폼과의 데이터 연계 업무• 정형/반정형/비정형 데이터간의 데이터 정합성 검증• Data Lifecycle 에서 발생되는 비효율 업무 정의 및 개선 업무• 데이터 분석을 통한 데이터 탐색, 마트/지표 설계, 데이터모델링

![image](https://imgrec.cj.net/recruit/images/contents/b_f97.png)

지원자격• 5년 이상의 데이터 엔지니어링 실무 경험• 클라우드 환경 기반의 데이터 파이프라인 설계 및 ETL 구축/운영 역량을 갖추신 분• Kafka 기반 실시간 데이터 파이프라인 구축/운영 역량을 갖추신분• RestAPI 를 이용한 Bulk 데이터 수집 / 전송 개발 경험 있으신분• 이기종간 데이터 통합 및 실시간 데이터 파이프라인 설계/구축 경험이 있으신분• Python, Spark, Airflow, SQL 등 대용량 데이터 처리 및 파이프라인 구현이 가능하신분• 전사의 다양한 이해관계자들과 협업하고 복잡한 데이터 엔지니어링 문제를 이해하기 쉽게 전달할 수 있는 커뮤니케이션 능력을 갖추신분

우대사항• Spark, Flink 등 대용량/실시간 처리를 위한 분산/병렬 시스템 개발 경험 있으신분• 비즈니스 도메인 지식을 빠르게 이해하고, 데이터 기반 의사결정을 위한 데이터 엔지니어링 경험이 있으신분• 데이터 정합성보장 및 파이프라인 효율성을 높이기 위한 개선 경험이 있으신분• CI/CD, K8S 등 데이터 인프라 관련 실무 경험과 역량이 뛰어나신분• AWS, Google Clould 등 퍼블릭 클라우드 플랫폼 기반 데이터 인프라 구축 경험 있으신분
