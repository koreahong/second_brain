### 카카오(KAKAO)∙경기 성남시∙경력 3-5년[다음설립준비법인] Data Engineer

## 포지션 상세

본 공고는 카카오가 아닌 분사 예정 법인인 "다음설립준비법인" 영입 공고입니다.

[회사소개]

다음설립준비법인은 카카오 그룹사 소속으로, 많은 분들에게 익숙한 다음(Daum) 서비스를 제공하는 회사입니다. 다음설립준비법인은 Daum이 쌓아온 역사를 바탕으로, 더 풍부하고 다양한 콘텐츠를 통해, 새로운 New Daum을 만들어 나갈 계획입니다. 이 흥미진진한 여정에 동참할 열정 넘치는 분들을 기다리고 있습니다. 함께 꿈꾸고 함께 만들어가는 새로운 여정에 여러분을 초대합니다!

[조직소개]

다음사업팀은 포털의 주요 매출원인 광고사업과 제휴사업을 담당하는 조직입니다. 광고사업은 여러가지 광고시스템을 안정적으로 연동하고 광고상품의 효율화를 진행하여 매출 성장을 이끕니다. 제휴사업은 트래픽과 매출 성장을 위해서 외부 파트너사와 제휴를 추진하고 새로운 사업모델을 개발합니다.

주요업무• 포털서비스와 광고의 지표 데이터를 수집 및 가공• 지표 기반의 코호트/퍼널 분석 시스템 개발• 통합 대시보드용 API 개발, BI 도구용 데이터 처리 등을 수행

자격요건• 컴퓨터공학, 통계학, 전산학 등 관련 전공에 대한 기본적인 이해• Python / Java 기반 데이터 처리, REST API 개발• 데이터 파이프라인 구축 경험 (ETL/ELT 설계 및 운영)• SQL 및 DW(BigQuery, Hive 등) 활용 개발• Grafana, Tableau, Metabase 등 BI 도구 중 하나 이상 사용 경험

우대사항• 광고 및 서비스 지표 데이터 분석 경험 보유• 코호트/퍼널 지표 설계 및 집계 자동화 경험 보유• Apache Airflow, Spark, Kafka 등 데이터 처리/스케줄링 도구 경험• DW/BI용 데이터 마트 설계 경험 보유
