ADA 소개

ADA 는 “데이터와 AI 로 고객사의 비즈니스 혁신(트랜스포메이션)을 돕는다”는 비전 하에 동남아시아, 인도, 한국, 일본의 12개 시장의 1500여개 유수한 고객사들에게 “컨설팅 – 구축 – 운영”의 토탈 서비스를 제공하는 컨설팅 파트너입니다. Softbank Vision Fund 및 동남아시아 선도 통신그룹인 Axiata, 일본의 대형 그룹사 Sumitomo, Mitsui 등으로부터 투자를 지속적으로 받아, 사업 영역 및 마켓을 확장하고 고속 성장하고 있으며, 데이터와 AI, 마케팅, 전자상거래, 메신저기반 고객응대 서비스 플랫폼에 전문성을 갖고 있는 1400여명의 임직원이 글로벌로 긴밀하게 협업하면서 고객사의 비즈니스를 돕고 있습니다.

ADA Korea 는 CDP(고객데이터플랫폼) 컨설팅/구축/운영 선도 업체로서 (2023년도에 트레저데이터 코리아 팀과 합병) 국내 CDP 태동기인 2019년부터 대기업/중견기업의 CDP 파트너로서 국내 최다 최고 수준의 레퍼런스를 보유하고 있습니다.CDP의 활용에도 기본적으로 포함되지만 CDP 보유와 활용 여부와 별개로, CX (고객경험)및 DX (디지털 트랜스포메이션)영역 전반에 걸쳐 데이터를 분석해서 실행 가능 인사이트를 찾아내고, ML과AI를 활용해서 개인화 혹은 최적화 알고리즘을 설정하고, 이를 토대로 각 비즈니스 분야에서 활용되는 다양한 애플리케이션들과 연동 제어하는 전반에 걸쳐 다양한 첨단 사례와 노하우를 갖고 있습니다.

모시는 글

ADA Korea 의 Data Solution 조직은 CDP 에 대한 역량과 기술 뿐만 아니라, 데이터를 기반으로 ActionableInsight를 도출하고, 실제 Action 까지 맡아서 진행합니다. 고객사를 위한 Data Analysis, Data Product, Data Action 을 풀싸이클로 실행하는 것이 Data Solution 조직의 일상이며, 컨설턴트, 비즈니스&데이터 분석가, 데이터사이언티스트, 데이터엔지니어, CRM 등 개인화운영자, PM 각 역할의담당자들이 한 팀으로구성되어고객과호흡을맞추고일을진행합니다.

이와 같은 특징의 가치 및 제공 방식이 시장에서 점점 인정을 받으면서 팀을 확장하고 있기에, 팀의 중요한 멤버인 데이터 솔루션 엔지니어를 모십니다. 팀이 제공하는 가치가 융복합적이다보니전형적인 ETL 작업 중심의 데이터 엔지니어보다는 다소 확장된 역량을 갖고 계신 인재분들을 우대합니다만, 무엇보다도 일과 팀에 대한 자세 및 커뮤니케이션, 그리고 기본 역량이 출중하신 분들을 소중하게 모시고 싶습니다. 나머지는 강력한 팀 역량과 팀웍을 통해, 그리고 이 분야에서 24년의 경험을 지닌 저도 함께 보완해 드리고 성장시켜드릴 수 있습니다.

ADA Korea 대표 / 그로스해커 & 데이터사이언티스트고영혁.

주요 업무

- ETL 파이프라인구축 – 시스템 및 애플리케이션사이의데이터수집/정제/전송
- 분석용 데이터 마트 설계 및 구축 – 비즈니스 및 분석 요건을 충족시키는 데이터 마트
- 차트 및 대시보드 설계 및 구축 - 비즈니스 및 분석 요건을 충족시키는 차트 및 대시보드
- 데이터 생성 - 웹 태깅, 서버 사이드 로그인/가입 이벤트 처리
- API를 활용한 애플리케이션 연동 - 다른 애플리케이션(CRM, 웹/앱 개인화 도구, 메신저, 광고 시스템 등)과의 연동과 자동화
요구 사항

- AWS (EC2, RDS, Lambda, API endpoint 등) 활용경험 (혹은이에준하는 Azure 및 GCP 활용경험)
- REST API 생성/운영또는준비되어있는 API 활용경험
- SQL 및 Python활용한 데이터 분석 마트와 자동화 파이프라인 생성 경험
- 인사이트 추출을 위한 BI 대시보드 설계 구축 경험
- 웹서비스 개발 경험 또는 웹 (혹은 모바일 앱) 태깅 구축 경험
요구 스킬

- AWS (API gateway,Lambda,S3,EC2) 혹은이에준하는 Azure 및 GCP
- SQL
- Python, Jupyter Notebook
- HTML, Javascript, CSS
- WebServer (JSP,PHP,ASP 등)
- GTM (Google Tag Manager) 또는그에준하는태그매니저관련경험
우대 사항

- Databricks, Presto(Trino), Hive
- 데이터분석, ML, RAG
- Windows/Mac/Android/iOS앱개발 (OS 종류 상관 없음)
- C/C++
- DevOps 경험
- 서비스 가입 및 로그인 부분 서버 모듈의 구축 경험
